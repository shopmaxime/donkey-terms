# Donkey Terms

A small Agent Skill that turns complex text, ideas, errors, proposals, or technical explanations into concise plain language.

It outputs:

- a one-sentence short version
- a few plain-language bullets
- why it matters

## Install

### Agent Skills directory

Submit or install this repository as an Agent Skill using the `SKILL.md` file at the repo root.

### Pi

```bash
pi install git:github.com/shopmaxime/donkey-terms
```

Or copy `SKILL.md` into your Pi skills directory:

```bash
mkdir -p ~/.pi/agent/skills/donkey-terms
cp SKILL.md ~/.pi/agent/skills/donkey-terms/SKILL.md
```

## Use

Ask for something in "donkey terms", or ask to simplify, summarize, de-jargon, or explain something for a non-expert.
